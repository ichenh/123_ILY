# ✨ 123 I ❤️ U：手势交互式 3D 粒子告白墙

一个简单、免配置的网页项目，展示了基于摄像头手势控制的 **实时 3D 粒子效果**。

-----

## 🚀 快速开始

本项目无需任何服务器环境或复杂的构建步骤，下载后即可在现代浏览器中运行。

1.  **克隆或下载**该项目到本地。
2.  双击打开 **`index.html`** 文件即可直接运行。
3.  首次打开时，请根据提示授权摄像头访问权限。

## 📂 目录结构

| 文件/文件夹 | 描述 |
| :--- | :--- |
| **`index.html`** | 项目的主入口页面，包含 HTML 结构和主要的 3D 渲染画布。 |
| **`script.js`** | 项目的核心 JavaScript 文件，包含 **Three.js 场景设置**、**手势处理逻辑** 和 **粒子动画**。 |
| **`style.css`** | 样式文件，用于控制页面布局、摄像头视频流及全屏按钮的样式。 |
| **`models/`** | **非必要文件夹**。仅作为 **CDN 故障转移备用**。如果用户所在地区无法访问 MediaPipe 模型的 CDN 资源，项目将自动从该本地文件夹加载模型。 |

## 💡 核心功能与手势控制

本项目通过 MediaPipe 实时识别手部姿势，将手指数量映射为 3D 粒子文字。

| 手势 | 手指数量 (Count) | 场景效果 | 描述 |
| :--- | :--- | :--- | :--- |
| **握拳** | 0 | 随机粒子 | 场景重置为默认的随机粒子背景。|
| **食指** | 1 | 数字 "1" | 粒子组成数字 **"1"**。|
| **食指 + 中指** | 2 | 数字 "2" | 粒子组成数字 **"2"**。|
| **三指 (3)** | 3 | 数字 "3" | 粒子组成数字 **"3"**。|
| **五指张开** | 5 | **I ❤️ U (告白模式)** | 持续保持 100ms 后，粒子爆炸式重组为 "I ❤️ U" 心形，并伴随爱心跳动和气球升空动画。|

## ⚙️ 技术栈

  * **Three.js**: 负责高性能的 **3D 渲染**、**粒子系统**的创建、更新和动画效果（包括爱心跳动和气球）。
  * **MediaPipe Hand Landmarker**: 负责 **实时的手势识别** 和 **手部关键点追踪**。
  * **ES Modules**: 用于模块化管理代码和依赖。
  * **CDN 故障转移机制**: 项目内置逻辑，确保在网络环境受限时，能自动切换到加载本地 `models/` 文件夹中的模型资源，保证项目可用性。
